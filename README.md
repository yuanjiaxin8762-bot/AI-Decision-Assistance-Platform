# AI Persona Decision Assistant  
# AI 多人格决策辅助系统

**Persona Switch** 是该项目的前端产品名称，基于生成式 AI 的多角色决策辅助系统。

---

## 项目简介 | Project Overview

AI Persona Decision Assistant 是一个基于生成式 AI 的**多角色决策辅助系统**。该系统通过模拟不同人格特征的 AI Agent，从多个视角对用户问题进行分析，并生成多种可能的决策路径，从而帮助用户在复杂情境中进行更加理性和全面的思考。

在现实生活中，人们在面对重要选择时往往容易受到情绪、经验或单一视角的影响，这会导致决策偏差。该系统通过构建多个具有不同思维方式的 AI 人格角色，使同一个问题能够被多种思考方式重新解读，从而为用户提供更加丰富的决策参考。

与传统 AI 问答系统不同，本项目**并不直接给出单一答案**，而是通过多角色思考机制，帮助用户理解不同选择背后的逻辑与潜在影响。系统希望让 AI 从「答案生成工具」转变为「思考框架生成工具」，成为用户在复杂问题中的思考伙伴。

---

## 项目背景 | Background

在当代社会中，个体面临的决策复杂度正在不断提升。职业路径的不确定性、社交关系的多样化以及信息环境的高度开放，使得用户在日常生活中需要频繁做出重要选择。然而，大多数决策往往依赖个人经验与直觉，这种单一视角容易导致判断偏差；在压力或情绪波动的情境下，人们也更容易做出冲动决策。

心理学研究表明，人类决策往往受到情境与情绪的影响，而不同人格特征也会产生不同的判断方式。因此，如果能够模拟多种思维模式对同一问题进行分析，用户就能够从更多视角理解问题，从而获得更加全面的决策参考。随着生成式 AI 与大语言模型的发展，构建具有稳定人格特征的 AI Agent 已经成为可能。本项目正是在这一背景下，探索 AI 人格模拟技术在决策辅助领域中的应用可能性。

---

## 项目目标 | Project Goal

构建一个基于 AI 人格模拟的**多角色决策辅助系统**，通过生成多个具有不同思维特征的 AI Agent，为用户提供多视角思考路径。系统通过对同一问题进行多角色分析，使用户能够比较不同观点之间的差异，并理解每种决策路径背后的逻辑，从而在复杂情境中做出更加理性和清晰的判断。

## 系统核心功能 | Features

### 1. 用户人格建模

系统通过结构化问题采集用户的基础信息与人格特征，并构建用户人格模型。建模过程包括：

- **基础信息采集**：昵称、年龄、职业、教育、地理位置、自述等
- **人格特征分析**：五维人格罗盘（理性、冒险、利他、长期、自主）与维度微调
- **价值观识别**：价值观排序与人生原则陈述
- **关键经历记录**：经历锚点选择与时间轴
- **情境测试验证**：决策风格情境题与理由补充

这些信息将被整合为用户的人格向量，用于生成 AI 人格角色。

### 2. AI 人格生成

在用户人格模型的基础上，系统生成多个具有不同思维风格的 AI Agent，例如：

- **理性型 Agent**：以逻辑分析与风险评估为核心，强调长期规划
- **情绪型 Agent**：关注情感因素与人际关系，提供共情视角
- **冒险型 Agent**：强调机会与突破，倾向于高风险高收益决策
- **感性型 / 谨慎型**：补充直觉与底线思维视角

这些 Agent 会在用户人格基础上进行不同方向的思维扩展（偏移可调）。

### 3. 多视角问题分析

当用户提出问题后，不同 AI Agent 会分别从各自思维方式出发进行分析，内容包括：问题理解、决策逻辑、风险评估、结果预测。同一问题可被多种思考方式重新解读。

### 4. 决策结果对比

系统将不同 Agent 的分析结果进行结构化展示，并以可视化形式呈现（观点聚类、风险热力图、决策路径树等）。用户可以对比不同决策建议、风险等级、长期影响与情绪影响。

### 5. 决策辅助输出

系统最终不会直接替用户做出决定，而是通过多视角分析为用户提供思考参考。用户可以在不同观点之间进行比较，并结合自身价值观做出最终判断。

## 技术实现思路 | Technical Approach

整体流程：

```
用户输入问题 → 用户人格建模 → AI Persona 生成 → 多 Agent 推理 → 结果对比 → 辅助决策输出
```

涉及技术：Large Language Model（LLM）、Persona Modeling、Multi-Agent System、Decision Visualization。当前实现中，前端通过后端代理调用 DeepSeek API，完成多 Agent 对话与决策分析。

## 项目意义 | Project Value

本项目尝试探索 AI 在决策辅助领域中的新应用方式。通过构建多角色思考系统，AI 不再仅仅提供答案，而是帮助用户看到更多可能性，从而：

- 拓展思考维度
- 减少情绪化决策
- 理解不同选择的逻辑
- 提升复杂问题的判断能力

最终使 AI 成为用户在面对复杂问题时的**思考伙伴**。

## 快速开始 | Quick Start

### 运行（推荐）

已内置后端代理接口，前端会调用 `/api/chat`。请按下面步骤启动：

```bash
# 1) 安装依赖
pip install fastapi uvicorn httpx

# 2) 设置 DeepSeek Key（PowerShell 当前窗口有效）
$env:DEEPSEEK_API_KEY="你的DeepSeekKey"

# 3) 启动服务（同时托管前端静态页面 + 后端代理）
uvicorn server:app --host 127.0.0.1 --port 8000 --reload
```

浏览器访问：**http://127.0.0.1:8000**

### 说明

- `server.py` 提供后端代理：`POST /api/chat`，支持从环境变量或项目根目录 `.env.local` 读取 `DEEPSEEK_API_KEY`
- 前端 `app.js` 不暴露 Key，请求经本地代理转发至 DeepSeek
- 若后端未启动或 Key 未配置，对话会提示连接失败，请检查配置后重试

## 当前实现概览

- **人格孵化室**：六步流程（基础信息、人格罗盘、价值观、经历锚点、情境测试、Agent 定制）、五维人格罗盘、关键经历锚点、五类 Agent 偏移预设
- **多 Agent 对话**：在场感脉冲、辩论按钮、@指定 Agent、追问/发送，回复结合用户六步画像与 Agent 设定
- **决策矩阵**：观点聚类、风险热力图、决策路径树、报告导出入口
