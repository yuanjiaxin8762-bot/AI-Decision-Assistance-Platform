## 步骤切换与对齐

**切换步骤** · `app.js` 约 1209–1238 行 `setIncubatorStep(step)`  
- 设置 `incubatorState.currentStep`  
- 为 `.incubate-step` 切换 `active`（仅当前 `data-step` 显示）  
- 更新进度条、步骤文案、上一步/下一步/完成/跳过按钮显隐  
- 调用 `updateStepChips()`、`updateSeedPreview()`、`scheduleIncubatorAlignment()`

**步进入口** · `app.js`  
- `btn-next-step` → `onNextStep()`  
- `btn-prev-step` → `onPrevStep()`  
- `btn-skip-step` → `onSkipStep()`  
- `btn-complete-incubate` → `completeIncubation()`  
- `.step-chip` 点击 → `setIncubatorStep(Number(chip.dataset.jumpStep))`

**初始化** · `app.js` 约 2102–2156 行 `initIncubator()`  
依次调用：`bindBaseProfileActions`、`bindTraitInputs`、`bindRadarDrag`、`bindValueCompass`、`bindExperienceActions`、`bindTimelineActions`、`bindScenarioActions`、`bindAgentCustomizer`，以及各 `render*`，最后 `setIncubatorStep(1)`。
